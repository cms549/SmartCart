What server has to do:
	1) /occupancy 
		get returns the current occupancy 
		post (increments the occupancy)
		/leave
		decrements occupancy by one
		
	2) /topics
		get returns list of topics
		post search for specific topic
		
	3) /items
		get returns list of all items
		post returns list of all items matching search keyword or id number
			checks in form for:
				idnum
				keyword
